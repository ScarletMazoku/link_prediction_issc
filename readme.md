# ITMO Practice Spring 2026

Исследовательский проект по задаче **link prediction** в научных коллаборациях на основе графовых данных.

## О проекте

Цель проекта — предсказать вероятность будущей коллаборации между авторами научных публикаций.

В качестве данных используется конференция **ICCS (International Conference on Computational Science)**.  
Были собраны публикации за весь период (с 2002 года), после чего построен граф соавторства.

---

## Постановка задачи

Задача формулируется как **link prediction**:

- узлы графа — авторы;
- рёбра — факт совместной публикации;
- мультиграф — один и тот же набор авторов может иметь несколько совместных работ.

Цель модели — предсказать вероятность появления нового ребра (будущей коллаборации).

---

## Построение графа

Граф строится следующим образом:

- для каждой статьи берётся список авторов  
- для всех пар авторов создаются рёбра  
- если авторы публиковались несколько раз — создаются множественные рёбра  

---

## Модель

Реализована модель для предсказания вероятности связи между авторами.

**Вход:**
- пара узлов (авторов)  
- признаки, полученные из графа  

**Выход:**
- вероятность коллаборации
